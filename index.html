Proyectos

Tech Shop

Este proyecto es una modernización de una antigua tienda online de hardware informático creada originalmente alrededor de 2010. El objetivo fue reconstruir completamente la página de catálogo desde cero utilizando una estructura semántica moderna de HTML5 y CSS responsive, sustituyendo técnicas obsoletas como el posicionamiento absoluto, los diseños fijos y el uso de CSS no estándar y Tailwind.

Clon de Spotify

Proyecto de maquetación móvil y desktop con menu hamburquesa desplegable en versopn de pantalla para moviles inspirado en la marca Spotify, con un menú interactivo sidebar en la versión móvil. La maquetación general de los elementos principales se hizo con CSS Grid, mientras que los elementos hijos se organizaron con Flexbox.

Recipe page

Este proyecto consiste en el desarrollo de una página de receta responsive basada en un diseño proporcionado, que incluye una imagen principal, una descripción de la receta, una sección de tiempo de preparación, una lista de ingredientes, instrucciones paso a paso y una tabla de información nutricional. El objetivo principal fue trabajar con una estructura semántica en HTML5 y aplicar estilos precisos utilizando CSS.
